# 웹 계산기 프로젝트

간단한 웹 계산기 앱입니다.  
HTML, CSS, JavaScript로 제작되었으며, 버튼 클릭으로 숫자와 연산을 입력하고 계산할 수 있습니다.

## 🖥️ 사용 방법
1. index.html 파일을 브라우저에서 열기
2. 숫자와 연산 버튼 클릭
3. '=' 버튼 클릭 시 결과 표시
4. 'C' 버튼 클릭 시 초기화

## 🎨 특징
- 깔끔한 다크 테마 디자인
- 버튼 클릭 애니메이션 적용 가능
- GitHub Pages로 배포 가능
